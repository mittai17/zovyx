# Zuvix Codex

Official Zuvix provider and harness plugin for OpenAI Codex app-server integration. It exposes the Codex-managed GPT model catalog and the Codex runtime surfaces used by Zuvix agents.

Install from Zuvix:

```bash
zuvix plugin add @zuvix/codex
```

Use this plugin when you want Zuvix to run Codex-backed model turns, media understanding, and prompt overlays through the Codex app-server harness.
