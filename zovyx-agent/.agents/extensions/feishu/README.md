# Zuvix Feishu/Lark

Official Zuvix channel plugin for Feishu and Lark workplace chats. Community maintained by @m1heng.

Install from Zuvix:

```bash
zuvix plugin add @zuvix/feishu
```

Configure the Feishu/Lark app credentials in Zuvix, then connect the plugin to the chats where agents should receive and send messages.
