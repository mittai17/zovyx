# Qwen plugin

Adds Qwen, Qwen Cloud, Model Studio, DashScope, Qwen Oauth, Qwen Portal, Qwen CLI model provider support to Zuvix.

## Distribution

- Package: `@zuvix/qwen-provider`
- Install route: included in Zuvix

## Surface

providers: qwen, qwencloud, modelstudio, dashscope, qwen-oauth, qwen-portal, qwen-cli; contracts: mediaUnderstandingProviders, videoGenerationProviders
