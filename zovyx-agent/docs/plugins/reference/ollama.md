# Ollama plugin

Adds Ollama, Ollama Cloud model provider support to Zuvix.

## Distribution

- Package: `@zuvix/ollama-provider`
- Install route: included in Zuvix

## Surface

providers: ollama, ollama-cloud; contracts: memoryEmbeddingProviders, webSearchProviders
